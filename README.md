# Greetings App

A simple full stack greeting card web application where users can create personalized greeting cards with custom templates, messages, and profile pictures.

The project includes authentication, image upload, premium templates, live preview, and card download functionality.

---

## Features

- User Signup & Login
- JWT Authentication
- Persistent Login
- Upload Profile Picture
- Dynamic Greeting Card
- Multiple Templates
- Category Filtering
- Premium Template Popup
- Live Card Preview
- Download Greeting Card as Image
- Responsive UI

---
